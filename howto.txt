This is a command-line javaScript app that will find the differences between two JSON objects. It requires installation of NodeJs LTS runtime. To install it for MacOS, Linux, and Windows you can refer to: 
https://nodejs.dev/learn/how-to-install-nodejs. 
To run the app, type : 
node app.js file_a.json file_b.json.
Both JSON files file_a.json and file_b.json have to be in the same directory with main app.
